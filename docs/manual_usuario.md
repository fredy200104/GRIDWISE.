# Manual de Usuario GridWise (Residencial)

## 1. Inicio de sesion y registro
- Crea una cuenta con correo, contrasena y telefono.
- Inicia sesion con correo/contrasena o Google.
- Si olvidas la contrasena, usa la recuperacion por correo o por telefono registrado.

## 2. Dashboard
- Visualiza consumo diario y mensual en kWh.
- Consulta costo estimado diario y tendencia de consumo.
- Identifica picos y alertas de consumo elevado.

## 3. Dispositivos
- Agrega dispositivos del hogar con nombre, tipo, ubicacion y uso diario.
- Edita o elimina dispositivos segun sea necesario.
- Activa o desactiva dispositivos para reflejar su estado real.

## 4. Reportes
- Revisa consumo diario (24h), semanal y mensual.
- Consulta maximos, minimos, costo estimado y CO2 estimado.
- Actualiza los datos con gesto de refrescar.

## 5. Alertas
- Recibe alertas cuando el consumo supera el umbral configurado.
- Marca alertas como leidas individualmente o todas a la vez.

## 6. Recomendaciones
- Consulta recomendaciones automaticas basadas en patron de consumo.
- Aplica acciones de ahorro sugeridas para reducir costo y picos.

## 7. Conexion IoT (simulada)
- Vincula dispositivos IoT simulados desde el boton "Vincular".
- Cambia estado conectado/desconectado por dispositivo.
- Ejecuta simulacion para generar actualizaciones de potencia en tiempo real.
- Mantener presionado un dispositivo para eliminarlo.

## 8. Perfil y preferencias
- Edita nombre y telefono.
- Configura tarifa por kWh y umbral de alertas.
- Activa o desactiva notificaciones in-app.
