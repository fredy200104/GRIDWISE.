# Politicas de privacidad - GridWise

Fecha de actualizacion: 2026-04-24

## 1. Alcance
GridWise es una aplicacion residencial para monitoreo y gestion de consumo energetico del hogar.

## 2. Datos recopilados
- Datos de cuenta: nombre, correo y telefono.
- Datos operativos: dispositivos registrados y configuraciones de consumo.
- Datos de telemetria: consumo energetico y estado IoT.

## 3. Finalidad del uso
Los datos se usan para:
- Autenticacion y acceso seguro.
- Visualizacion de consumo y reportes.
- Generacion de alertas y recomendaciones.
- Integracion con dispositivos IoT.

## 4. Base de seguridad
Se aplican controles de seguridad como autenticacion con Firebase, reglas de acceso en Firestore, validacion de token en backend y control de dispositivos IoT por token.

## 5. Comparticion
No se comercializan ni comparten datos personales con terceros, salvo servicios tecnicos necesarios para operar la plataforma.

## 6. Conservacion
Los datos se conservan durante la vida util de la cuenta o mientras sean necesarios para objetivos academicos del prototipo.

## 7. Derechos del usuario
El usuario puede solicitar correccion, actualizacion o eliminacion de sus datos.
